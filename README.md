# 🎬 Movie Recommendation System

A Machine Learning-based Movie Recommendation System that suggests movies based on similarity using content-based filtering techniques.

---

## 📌 Overview

This project builds a recommendation system that suggests movies similar to a selected movie by analyzing features such as genres, keywords, and metadata.

Recommendation systems are widely used in platforms like Netflix and Amazon to enhance user experience through personalized content suggestions.

---

## 🎯 Objective

- Build a movie recommendation system using machine learning  
- Understand recommendation algorithms and similarity techniques  
- Apply data preprocessing and feature engineering  
- Generate personalized movie suggestions  

---

## ⚙️ Tech Stack

- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Jupyter Notebook  

---

## 📊 Dataset

- Contains movie metadata such as:
  - Movie titles  
  - Genres  
  - Keywords  

---

## 🧠 Methodology

### 1️⃣ Data Preprocessing
- Cleaned dataset  
- Handled missing values  
- Structured data for modeling  

### 2️⃣ Feature Engineering
- Combined important features (genres, keywords)  
- Converted text into vectors using CountVectorizer / TF-IDF  

### 3️⃣ Similarity Calculation
- Used **Cosine Similarity** to measure similarity between movies  

### 4️⃣ Recommendation System
- Input: Movie name  
- Output: Top similar movies  

---

## 🚀 Features

- Recommend movies based on similarity  
- Fast and efficient recommendation system  
- Uses real-world dataset  
- Simple and scalable ML pipeline
