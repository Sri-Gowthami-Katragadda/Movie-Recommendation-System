# 🎬 Movie Recommendation System

A Machine Learning-based system that recommends movies based on similarity using content-based filtering.

---

## 📌 Overview
This project suggests movies similar to a given movie by analyzing features like genres and metadata using cosine similarity.

---

## ⚙️ Tech Stack
- Python  
- Pandas, NumPy  
- Scikit-learn  

---

## 🧠 Method
- Data preprocessing  
- Feature engineering  
- Cosine similarity for recommendations  

---

## 🚀 Features
- Recommends similar movies  
- Fast and efficient  
- Simple ML pipeline
